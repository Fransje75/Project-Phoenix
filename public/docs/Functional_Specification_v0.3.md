# Project Phoenix — Functional Specification v0.3

## Hoofdflow
1. Game type kiezen
2. Manager(s) instellen
3. Club kiezen of carrière zonder club starten
4. Football Today
5. Taken, mail en kalender afhandelen
6. Team, training, transfers, financiën en infrastructuur beheren
7. Tijd voortzetten tot het volgende relevante event

## Fidelity-eisen
- Snelheid en eenvoud behouden
- Hidden attributes verborgen houden
- Football Today als commandocentrum
- Informatie en advies zonder automatische beslissingen
