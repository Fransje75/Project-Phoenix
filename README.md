# Project Phoenix Command Center v1.7

## Nieuwe functionaliteit
- Volwaardige Executable Explorer
- Module Registry
- Function Registry
- Structure Registry
- Address Register
- Cross references
- Evidence, notes en open questions per functie
- Koppelingen naar tests en savegames
- Reverse Engineering Status per engine
- Zoeken en filteren op recordtype
- Modulaire `data/executable/`-structuur

## Initiële research records
- `UEFA2000b.exe`
- DirectDraw runtime
- BltFast call path
- Savegame write coordinator
- Training profile apply
- Match event generator
- Player record
- Training schedule
- Match event structure

Deze records zijn duidelijk gemarkeerd als verified, hypothesis, planned of unresolved. Er zijn geen onbekende adressen als feiten ingevuld.

## Installeren
1. Pak de ZIP uit.
2. Vervang via GitHub Desktop de huidige PPCC-bestanden.
3. Laat de verborgen `.git`-map staan.
4. Commit: `Add PPCC v1.7 Executable Explorer`
5. Push origin.
6. Controleer Actions en de live site.

Geen npm, dependencies of buildstap nodig.
