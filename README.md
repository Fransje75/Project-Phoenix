# Project Phoenix Command Center v1.1

Productieklare React + TypeScript + Vite-versie van het centrale Project Phoenix-onderzoeksplatform.

## Lokaal testen
```bash
npm ci
npm run build
npm run dev
```

## GitHub Pages
Upload de inhoud van deze map naar de root van je repository. Maak zo nodig handmatig `.github/workflows/deploy.yml` aan op GitHub. Kies daarna bij **Settings → Pages → Source** voor **GitHub Actions**.

## Gecontroleerd
- `npm ci`
- `npm run build`
- TypeScript build
- Vite productie-output
- relatieve GitHub Pages-paden
