# Project Phoenix Command Center v1.2

## Nieuwe functionaliteit
- Documentatie direct lezen in een ingebouwde reader
- Zoeken door alle Markdown-documenten
- Zoeken binnen het geopende document
- Zoekresultaten gemarkeerd in de reader
- Downloadoptie blijft beschikbaar
- Bijgewerkte Project Phoenix-documentatie v0.4

## Installeren
1. Pak de ZIP uit.
2. Vervang via GitHub Desktop alle PPCC-projectbestanden in je lokale repository.
3. Laat de verborgen `.git`-map staan.
4. Commit: `Add PPCC v1.2 documentation reader`
5. Push origin.
6. Controleer GitHub Actions en de live site.

Deze versie gebruikt nog steeds geen npm of buildstap.
